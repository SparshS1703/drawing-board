# Collaborative Drawing Board

real time collaborative drawing board where multiple users can draw together on a shared canvas, see each others cursers  and a global undo/redo functionality.

## Setup Instructions

## Prerequisites

-Node.js
-npm
-bash terminal

## Steps

git clone 
https://github.com/SparshS1703/drawing-board.git
npm install
npm start


server will start on 
https://localhost:3000
